üöÄ How to Run & Test

Use Docker to run the server and the provided test scripts.

Start the Server (Docker)

# Build and start the server in detached mode
docker-compose up --build -d

Test Concurrency (Lab 2)

Run the concurrent test script. This will make 10 requests at once. A concurrent server should finish in ~1-2 seconds, while a single-threaded server would take ~10-11 seconds.

python test_concurrent.py

Test with the Client (Lab 1)

You can still use the Lab 1 client to test individual files:

# Test 1: Request the root page (which is index.html)
python client.py localhost 8080 / ./downloads

# Test 2: Download the PNG image (should save Dog.png)
python client.py localhost 8080 /Dog.png ./downloads

# Test 3: Test Directory Listing (should print HTML listing)
python client.py localhost 8080 /Books/ ./downloads
